怎么破解Typora？

想白嫖不想付费怎么办？Later这就教你！！！

官网

首先下载 Typora 的安装包 ：官网(国内可能打不开) 、中文站

选择你需要的版本下载，我下载了目前最新版 1.9.5

安装

我安装时忘截图了，简要说一下步骤吧

  • 双击安装包
  • 选安装路径（想改就改，暂时先记着）
  • 勾选 Create a desktop shortcut
  • 点击 Install 安装
  • 去除勾选 Launch Typora ，点击 Finish
